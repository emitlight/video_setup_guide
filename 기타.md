│   └── assets/
│       ├── index-Ci08f8VB.css   ← 모든 스타일
│       └── index-zfWvT8CT.js    ← 모든 React 코드

│   ├── App.tsx

# 사용 방법
바로 서빙 (빌드 없이): dist/ 폴더 내용만 웹서버에 올리면 바로 동작합니다.

소스 수정 후 재빌드:

bash 
npm install
(npm run build   # → dist/ 폴더 재생성)
npm run dev
npm vite --port 8099

콘텐츠 수정: src/lib/index.ts의 GUIDE_SECTIONS 배열에서 설정값, 체크리스트 항목, 프로젝트명 등을 직접 편집하세요!
